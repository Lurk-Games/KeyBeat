# KeyBeat
KeyBeat is a free-to-win Roblox rhythm game and it's now open source.

## Status

This project is under constant development, but we do our best to keep things in a stable state. This project will continue to evolve.

## Contributing

When it comes to contributing to the project, the two main things you can do to help out are reporting issues and submitting pull requests. Please refer to the [contributing guidelines](CONTRIBUTING.md) to understand how to help in the most effective way possible.

We love to reward quality contributions. If you have made a large contribution, or are a regular contributor, you are going to be in the game credits and probably get something special. If you have any questions, feel free to [reach out to us](mailto:help.vobibox@gmail.com) before doing so.
